Rent A Car
------------------------------------------------------------------------------------------------
- Ethereum Blockchain based POC to rent car with ether
- Project is built using Etherum Vanilla Blockchain, NodeJs, HTML5, JavaScript, CSS

Instructions
------------------------------------------------------------------------------------------------
- Install the node dependencies i.e. expressJS, socket.io
- Deploy the smart contract on testrpc/geth
- Get the smart contract mining address and change it in app.js
- Run the node instance and start using the project
